Introduction to Python - Complete Beginner Guide

Theory & Concepts

What is Python?

Python is a high-level, interpreted, general-purpose programming language created by Guido van Rossum in 1991. It emphasizes code readability with its notable use of significant indentation.

The Name "Python"

Python was NOT named after the snake! Guido van Rossum named it after the British comedy series "Monty Python's Flying Circus" which he enjoyed watching.

Key Features of Python

1. Easy to Learn and Read

  • Simple, English-like syntax
  • No semicolons or curly braces
  • Indentation-based structure

Example:

if age > 18:
    print("Adult")  # Indentation defines the block

2. Interpreted Language

  • Code runs line by line
  • No compilation step required
  • Immediate feedback and testing

Comparison:

  • Compiled (C, Java): Code → Compiler → Machine Code → Execution
  • Interpreted (Python): Code → Interpreter → Direct Execution

3. Dynamically Typed

  • No need to declare variable types
  • Type is determined at runtime
x = 5       # x is integer
x = "hello"  # now x is string (allowed!)

4. Object-Oriented

  • Everything is an object
  • Supports classes and inheritance
  • Encapsulation and polymorphism

5. Cross-Platform

  • Write once, run anywhere
  • Works on Windows, Mac, Linux
  • Same code, different platforms

6. Extensive Libraries

  • Over 300,000 packages on PyPI
  • Libraries for everything: web, data science, AI, automation

7. Free and Open Source

  • Completely free to use
  • Open source community
  • Anyone can contribute

  • Beautiful is better than ugly

Python vs Other Languages

Python vs Java

  • Python: Easier syntax, faster development, less verbose
  • Java: Faster execution, better for large enterprise apps

Python vs JavaScript

  • Python: Backend, data science, automation
  • JavaScript: Frontend, web interactivity, Node.js backend

Python vs C++

  • Python: Easier, slower execution, higher level
  • C++: Harder, faster execution, lower level, memory control

Rule of Thumb: Python trades some execution speed for development speed and readability.

Your First Python Program

The Traditional "Hello, World!"

print("Hello, World!")

That's it! Just one line. No imports, no class definitions, no main function.

Compare with Java:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Python's simplicity is clear!

How Python Code Executes

Source Code (.py file)
        ↓
    Python Compiler
        ↓
  Bytecode (.pyc file)
        ↓
 Python Virtual Machine (PVM)
        ↓
     Execution

Note: The compilation to bytecode happens automatically and is hidden from the user.
