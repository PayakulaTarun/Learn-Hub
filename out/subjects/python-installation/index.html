Python Installation & Setup - Step-by-Step Guide

Theory & Concepts

Why Install Python Locally?

While online Python interpreters exist, installing Python on your computer gives you:

  • Full control over versions and packages
  • Offline access - code without internet
  • Professional environment - same setup as industry
  • Better performance - no network delays
  • File system access - work with local files

Python Download Options

Official Python (Python.org)

Recommended for beginners

  • Latest stable version
  • Includes pip (package manager)
  • IDLE (basic IDE included)
  • Download: https://www.python.org/downloads/

Anaconda Distribution

Recommended for data science

  • Python + 250+ pre-installed packages
  • Jupyter Notebook included
  • Conda package manager
  • Larger download size (~500MB)
  • Download: https://www.anaconda.com/

Python Version to Install

Always install the latest Python 3.x version

  • Python 3.11 or 3.12 (as of 2024)
  • Python 2.x is obsolete (support ended 2020)
  • Check current version: https://www.python.org/downloads/

Installation Steps by Operating System

Windows Installation

Step 1: Download Installer

  1. Visit https://www.python.org/downloads/
  2. Click "Download Python 3.x.x" button
  3. File downloads: python-3.x.x-amd64.exe

Step 2: Run Installer

  1. Double-click the downloaded .exe file
  2. CRITICAL: Check "Add Python to PATH" ✅
  3. Click "Install Now" (installs Python + pip + IDLE)
  4. Wait for installation to complete
  5. Click "Close"

Why "Add to PATH" matters: Allows running python command from any folder in Command Prompt.

Step 3: Verify Installation Open Command Prompt (Win + R, type cmd) and run:

python --version

Should display: Python 3.x.x

If it shows Python 2.x.x or "not recognized", Python 3 isn't in PATH.

macOS Installation

Method 1: Official Installer (Recommended)

Step 1: Download

  1. Visit https://www.python.org/downloads/mac-osx/
  2. Download latest macOS installer
  3. File: python-3.x.x-macos11.pkg

Step 2: Install

  1. Double-click .pkg file
  2. Follow installation wizard
  3. Click "Continue" → "Install"
  4. Enter admin password
  5. Click "Close"

Step 3: Verify Open Terminal (Cmd + Space, type "Terminal") and run:

python3 --version

Note: Use python3 on macOS (not python)

Method 2: Using Homebrew

# Install Homebrew first (if not installed)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install Python
brew install python

Linux Installation

Most Linux distributions come with Python pre-installed. Check first:

python3 --version

If not installed or outdated:

Ubuntu/Debian:

sudo apt update
sudo apt install python3 python3-pip

Fedora:

sudo dnf install python3 python3-pip

Arch Linux:

sudo pacman -S python python-pip

Verifying Installation

Check Python Version

Windows:

python --version

macOS/Linux:

python3 --version

Check pip (Package Manager)

Windows:

pip --version

macOS/Linux:

pip3 --version

Should show: pip x.x.x from /path/to/pip

Launch Python Interactive Shell

Windows:

python

macOS/Linux:

python3

You'll see:

Python 3.x.x ...
>>>

Type exit() or press Ctrl+D to exit.

Setting Up Code Editor

Option 1: IDLE (Included with Python)

Pros: Pre-installed, simple, beginner-friendly Cons: Limited features

How to open:

  • Windows: Search "IDLE" in Start Menu
  • macOS: Applications → Python 3.x → IDLE
  • Linux: Type idle3 in terminal

Option 2: VS Code (Recommended)

Pros: Professional, free, extensions, debugging Cons: Requires setup

Installation:

  1. Download from https://code.visualstudio.com/
  2. Install VS Code
  3. Open VS Code
  4. Install Python extension:
    • Click Extensions icon (or Ctrl+Shift+X)
    • Search "Python"
    • Install official Microsoft Python extension

Option 3: PyCharm

Pros: Full-featured Python IDE, intelligent code completion Cons: Heavy, paid version for advanced features

Community Edition (Free): https://www.jetbrains.com/pycharm/download/

Option 4: Sublime Text, Atom, Notepad++

Lightweight alternatives for simple projects.

Writing Your First Program

Method 1: Using IDLE

  1. Open IDLE
  2. File → New File (Ctrl+N)
  3. Write code:
print("Hello, Python!")
  1. Save as hello.py
  2. Run → Run Module (F5)

Method 2: Using Command Line

  1. Create file hello.py with text editor
  2. Write:
print("Hello, Python!")
  1. Save file
  2. Open terminal/command prompt in same folder
  3. Run:
python hello.py    # Windows
python3 hello.py   # macOS/Linux

Method 3: Interactive Mode (REPL)

  1. Open terminal/command prompt
  2. Type python (or python3)
  3. Type code directly:
>>> print("Hello!")
Hello!

Good for quick testing!

Environment Variables and PATH

What is PATH?

PATH is a system variable that tells your OS where to find executable programs.

Why Add Python to PATH?

Without PATH, you must provide full path to Python:

C:\Users\YourName\AppData\Local\Programs\Python\Python312\python.exe

With PATH, just type:

python

Manually Add Python to PATH (Windows)

If you forgot to check "Add to PATH" during installation:

  1. Find Python installation path (usually C:\Users\YourName\AppData\Local\Programs\Python\Python3x)
  2. Right-click "This PC" → Properties
  3. Advanced System Settings → Environment Variables
  4. Under "System variables", select "Path" → Edit
  5. Click "New"
  6. Add Python path and Scripts folder:
    • C:\Users\YourName\AppData\Local\Programs\Python\Python3x
    • C:\Users\YourName\AppData\Local\Programs\Python\Python3x\Scripts
  7. Click OK, restart Command Prompt

Common Installation Issues

Issue 1: "python is not recognized"

Cause: Python not in PATH Solution:

  • Reinstall Python, check "Add to PATH"
  • Or manually add to PATH (see above)
  • Restart terminal after changes

Issue 2: Python opens Microsoft Store (Windows)

Cause: Windows redirects python command to Store Solution:

  1. Settings → Apps → Apps & Features → App execution aliases
  2. Turn OFF "App Installer python.exe" and "python3.exe"

Issue 3: Multiple Python Versions Installed

Symptom: Running python uses wrong version Solution:

  • Use specific version: python3.11 or py -3.11
  • Update PATH to prioritize desired version
  • Use virtual environments (covered later)

Issue 4: Permission Denied (macOS/Linux)

Solution: Use sudo for system-wide installation

sudo apt install python3

Issue 5: pip not working

Solution:

python -m pip --version  # Use python -m to invoke pip

Virtual Environments (Introduction)

Virtual environments isolate Python projects with separate packages.

Why use them?

  • Different projects need different package versions
  • Avoid conflicts between project dependencies
  • Keep global Python clean

Basic usage (will cover in detail later):

# Create virtual environment
python -m venv myproject_env

# Activate
# Windows:
myproject_env\Scripts\activate
# macOS/Linux:
source myproject_env/bin/activate

# Deactivate
deactivate
