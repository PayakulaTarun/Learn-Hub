Python Conditional Statements - Complete Guide

Theory & Concepts

What are Conditional Statements?

Conditional statements allow a program to make decisions and execute different code based on conditions.

Real-Life Analogy

Think of a traffic light:

  • IF light is green → Go
  • ELIF light is yellow → Slow down
  • ELSE (light is red) → Stop

Programs make similar decisions based on conditions.

Flow of Execution

Normal program: Executes line by line, top to bottom

With conditionals:

      [Start]
         ↓
    [Condition?]
      ↙     ↘
   True    False
     ↓       ↓
  [Code A] [Code B]
     ↓       ↓
      [Continue]

1. The if Statement

Purpose

Execute code only if a condition is True.

Syntax

if condition:
    # Code block (indented)
    # Executes only if condition is True

Flow Diagram

      [Check condition]
            ↓
      [Is it True?]
       ↙         ↘
    YES          NO
     ↓            ↓
  [Execute    [Skip code,
   code]      continue]
     ↓            ↓
      [Continue program]

Key Points

  1. Colon (:) is required after condition
  2. Indentation (4 spaces) defines the code block
  3. If condition is False, code block is skipped
  4. Program continues after the if block

Examples

age = 20

if age >= 18:
    print("You are an adult")
    print("You can vote")

print("Program continues")  # Always runs

Output:

You are an adult
You can vote
Program continues

If age was 15:

Program continues  # if block skipped

2. The if-else Statement

Purpose

Execute one block if condition is True, another block if False.

Syntax

if condition:
    # Code if True
else:
    # Code if False

Flow Diagram

      [Check condition]
            ↓
      [Is it True?]
       ↙         ↘
    YES          NO
     ↓            ↓
  [Code A]    [Code B]
     ↓            ↓
      [Continue program]

Exactly one block executes - never both, never neither.

Examples

age = 15

if age >= 18:
    print("You can vote")
else:
    print("You cannot vote yet")
    print(f"Wait {18 - age} more years")

Output:

You cannot vote yet
Wait 3 more years

3. The if-elif-else Statement

Purpose

Handle multiple mutually exclusive conditions.

Syntax

if condition1:
    # Code if condition1 is True
elif condition2:
    # Code if condition2 is True
elif condition3:
    # Code if condition3 is True
else:
    # Code if all conditions are False

Flow Diagram

     [Check condition1]
            ↓
       [True?]
      ↙      ↘
    YES       NO
     ↓         ↓
  [Code1]  [Check condition2]
     ↓         ↓
     |     [True?]
     |    ↙      ↘
     |  YES       NO
     |   ↓         ↓
     |[Code2]  [Check condition3]
     |   ↓         ↓
     |   |     [True?]
     |   |    ↙      ↘
     |   |  YES       NO
     |   |   ↓         ↓
     |   |[Code3]   [else Code]
     ↓   ↓   ↓         ↓
      [Continue program]

Key Points

  1. Only ONE block executes (first True condition)
  2. Once a condition is True, rest are skipped
  3. elif = "else if"
  4. else is optional (default case)
  5. Can have multiple elif blocks

Grade Example

score = 85

if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
elif score >= 70:
    grade = 'C'
elif score >= 60:
    grade = 'D'
else:
    grade = 'F'

print(f"Grade: {grade}")  # Output: Grade: B

Execution Flow:

  • score >= 90? False → Check next
  • score >= 80? True → Execute this block, skip rest
  • Result: grade = 'B'

4. Nested Conditional Statements

Purpose

Conditional inside another conditional for complex decision-making.

Syntax

if outer_condition:
    # Outer code
    if inner_condition:
        # Inner code (nested)
    else:
        # Inner else
else:
    # Outer else

Flow Diagram

      [Outer condition?]
         ↙      ↘
      TRUE     FALSE
        ↓         ↓
   [Inner     [Outer
  condition?]  else]
    ↙    ↘       ↓
  TRUE  FALSE    |
   ↓      ↓      |
 [Code1][Code2]  |
   ↓      ↓      ↓
     [Continue]

Examples

Example 1: Age and License Check

age = 20
has_license = True

if age >= 18:
    if has_license:
        print("You can drive")
    else:
        print("Get a license first")
else:
    print("Too young to drive")

Example 2: Login System

username = "admin"
password = "secret123"

input_user = input("Username: ")
input_pass = input("Password: ")

if input_user == username:
    if input_pass == password:
        print("Login successful!")
    else:
        print("Wrong password")
else:
    print("User not found")

When to Use Nested vs Multiple Conditions

Nested (dependent checks):

if is_logged_in:
    if has_permission:
        print("Access granted")

Second check only matters if first is True.

Multiple elif (independent checks):

if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'

Checks are mutually exclusive.

Common Patterns

1. Range Checking

if 0 <= score <= 100:  # Chained comparison
    print("Valid score")
else:
    print("Invalid score")

2. Multiple Conditions with and

if age >= 18 and has_id:
    print("Can enter")

3. Multiple Conditions with or

if is_student or is_senior:
    print("Discount applied")

4. Ternary Operator (One-liner if-else)

status = "Adult" if age >= 18 else "Minor"

# Equivalent to:
if age >= 18:
    status = "Adult"
else:
    status = "Minor"

Indentation Rules

Python uses indentation to define code blocks.

Correct Indentation

if condition:
    print("Line 1")  # 4 spaces
    print("Line 2")  # 4 spaces
    if nested:
        print("Nested")  # 8 spaces
print("Outside")  # 0 spaces

Indentation Errors

# ❌ Wrong: No indentation
if age > 18:
print("Adult")  # IndentationError!

# ❌ Wrong: Inconsistent indentation
if age > 18:
    print("Line 1")  # 4 spaces
      print("Line 2")  # 6 spaces - Error!

# ❌ Wrong: Tab mixed with spaces
if age > 18:
    print("Tab")   # Tab character
    print("Spaces")  # Spaces - Can cause issues!

Best Practice: Use 4 spaces (not tabs) for indentation.

Truthy and Falsy Values

Python converts values to boolean in conditions.

Falsy (treated as False):

  • False
  • None
  • 0, 0.0
  • Empty: '', [], (), {}, set()

Truthy (treated as True):

  • Everything else!

Examples

name = ""
if name:  # Empty string is falsy
    print("Hello", name)
else:
    print("No name provided")  # This runs

items = []
if items:  # Empty list is falsy
    print("Has items")
else:
    print("No items")  # This runs

count = 5
if count:  # Non-zero is truthy
    print("Count exists")  # This runs

Best Practices

1. Keep Conditions Simple

# ❌ Complex
if ((age >= 18 and has_license) or (age >= 16 and has_permit)) and not is_suspended:
    pass

# ✅ Clear with variables
can_drive_full = age >= 18 and has_license
can_drive_learner = age >= 16 and has_permit
is_allowed = (can_drive_full or can_drive_learner) and not is_suspended

if is_allowed:
    pass

2. Use Positive Conditions

# ❌ Double negative
if not is_invalid:
    pass

# ✅ Positive
if is_valid:
    pass

3. Avoid Deep Nesting

# ❌ Hard to read (3+ levels)
if a:
    if b:
        if c:
            if d:
                pass

# ✅ Early return/continue
if not a:
    return
if not b:
    return
if not c:
    return
if not d:
    return
# Main logic here

4. Order Conditions from Specific to General

# ✅ Correct order
if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
elif score >= 70:
    grade = 'C'

# ❌ Wrong order (always matches first)
if score >= 70:  # Matches 90!
    grade = 'C'
elif score >= 80:
    grade = 'B'  # Never reaches!
