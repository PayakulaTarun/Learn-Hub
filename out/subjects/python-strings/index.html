Python Strings - Complete Guide

Theory & Concepts

What is a String?

A string is a sequence of characters enclosed in quotes. Strings are immutable - once created, cannot be changed.

Real-Life Analogy

Think of strings like printed text on paper:

  • Once printed, you can't change the letters
  • You can read any character
  • You can make a copy with modifications
  • You can combine multiple pages

Same with Python strings - read-only, create new ones for changes.

String Characteristics

  1. Immutable: Cannot be changed after creation
  2. Sequence: Ordered collection of characters
  3. Iterable: Can loop through characters
  4. Indexed: Access characters by position
  5. Versatile: Many built-in methods

Creating Strings

Single vs Double Quotes

# Single quotes
name = 'Alice'

# Double quotes
greeting = "Hello, World!"

# Both are identical
text1 = 'Python'
text2 = "Python"
print(text1 == text2)  # True

When to use which:

  • Single quotes: General use
  • Double quotes: When string contains single quote
  • Triple quotes: Multiline strings

Quotes Inside Strings

# Double quotes allow single quotes inside
sentence = "It's a beautiful day"

# Single quotes allow double quotes inside
quote = 'She said "Hello"'

# Or use escape character
text = 'It\'s a beautiful day'
quote2 = "She said \"Hello\""

Triple Quotes (Multiline)

# Triple single quotes
multiline1 = '''This is line 1
This is line 2
This is line 3'''

# Triple double quotes
multiline2 = """This is line 1
This is line 2
This is line 3"""

# Preserves formatting
poem = '''
Roses are red,
Violets are blue,
Python is awesome,
And so are you!
'''

Escape Characters

| Escape | Meaning | Example | |--------|---------|----------| | \n | Newline | "Line1\nLine2" | | \t | Tab | "Name\tAge" | | \\ | Backslash | "C:\\Users" | | \' | Single quote | 'It\'s' | | \" | Double quote | "Say \"Hi\"" | | \r | Carriage return | (Windows line ending) |

print("First line\nSecond line")
print("Name\tAge\tCity")
print("C:\\Users\\Alice")

Raw Strings (r"")

Ignore escape characters:

# Normal string
path = "C:\\Users\\Alice\\new_folder"  # Need \\

# Raw string
path = r"C:\Users\Alice\new_folder"  # Cleaner!

regex = r"\d+\s+\w+"  # Useful for regex

String Indexing

Positive Indexing (0-based)

text = "Python"
#       012345

print(text[0])  # P (first character)
print(text[1])  # y (second character)
print(text[5])  # n (sixth character)
print(text[6])  # IndexError! (doesn't exist)

Negative Indexing

text = "Python"
#      -6-5-4-3-2-1

print(text[-1])  # n (last character)
print(text[-2])  # o (second from end)
print(text[-6])  # P (first character)

Visual:

String: P  y  t  h  o  n
Index:  0  1  2  3  4  5
Neg:   -6 -5 -4 -3 -2 -1

String Slicing

Syntax: string[start:stop:step]

text = "Python Programming"
#       0123456789...

# Basic slicing
print(text[0:6])    # Python (0 to 5)
print(text[7:18])   # Programming (7 to 17)
print(text[:6])     # Python (start to 5)
print(text[7:])     # Programming (7 to end)
print(text[:])      # Python Programming (full copy)

Slicing with Step

text = "Python"

# Every character
print(text[::1])   # Python

# Every 2nd character
print(text[::2])   # Pto (P, t, o)

# Reverse
print(text[::-1])  # nohtyP

# Reverse every 2nd
print(text[::-2])  # nhy (n, h, y)

Advanced Slicing

text = "abcdefghij"

print(text[2:8])     # cdefgh
print(text[2:8:2])   # ceg (every 2nd from 2 to 7)
print(text[8:2:-1])  # ihgfed (reverse from 8 to 3)
print(text[-3:])     # hij (last 3)
print(text[:-3])     # abcdefg (all except last 3)

String Immutability

Immutable: Cannot modify string after creation.

Cannot Modify Characters

text = "Python"

# ❌ This doesn't work
# text[0] = 'J'  # TypeError: str doesn't support item assignment

# ✅ Create new string
text = 'J' + text[1:]  # "Jython"
# OR
text = text.replace('P', 'J')  # "Jython"

Why Immutability Matters

original = "Hello"
modified = original.upper()  # Creates NEW string

print(original)  # Hello (unchanged)
print(modified)  # HELLO (new string)
print(id(original) == id(modified))  # False (different objects)

Benefits:

  • Thread-safe (no race conditions)
  • Can be used as dictionary keys
  • Hashable (can use in sets)
  • Memory efficient (string pooling)

String Methods

Python has 40+ string methods! Here are the most important:

Case Conversion

text = "Python Programming"

print(text.upper())      # PYTHON PROGRAMMING
print(text.lower())      # python programming
print(text.capitalize()) # Python programming (first letter)
print(text.title())      # Python Programming (each word)
print(text.swapcase())   # pYTHON pROGRAMMING (swap case)

Searching

text = "Hello, World! Hello!"

# find(): returns index or -1
print(text.find("World"))   # 7
print(text.find("Python"))  # -1 (not found)
print(text.find("Hello", 1)) # 14 (start from index 1)

# index(): returns index or raises ValueError
print(text.index("World"))  # 7
# print(text.index("Python"))  # ValueError!

# count(): count occurrences
print(text.count("Hello"))  # 2
print(text.count("l"))      # 5

Checking Content

# startswith() / endswith()
filename = "document.pdf"
print(filename.endswith(".pdf"))    # True
print(filename.endswith(".txt"))    # False
print(filename.startswith("doc"))   # True

# isdigit(), isalpha(), isalnum()
print("123".isdigit())      # True (all digits)
print("abc".isalpha())      # True (all letters)
print("abc123".isalnum())   # True (letters + digits)
print("  ".isspace())       # True (all whitespace)

# isupper(), islower()
print("HELLO".isupper())    # True
print("hello".islower())    # True
print("Hello".islower())    # False

Splitting and Joining

# split(): split by delimiter
text = "apple,banana,cherry"
fruits = text.split(",")  # ['apple', 'banana', 'cherry']

sentence = "Python is awesome"
words = sentence.split()  # ['Python', 'is', 'awesome'] (split by space)

# join(): join list into string
fruits = ["apple", "banana", "cherry"]
text = ", ".join(fruits)  # "apple, banana, cherry"

words = ["Python", "is", "awesome"]
sentence = " ".join(words)  # "Python is awesome"

Trimming

text = "   Hello, World!   "

print(text.strip())   # "Hello, World!" (both sides)
print(text.lstrip())  # "Hello, World!   " (left)
print(text.rstrip())  # "   Hello, World!" (right)

# Remove specific characters
text = "***Hello***"
print(text.strip("*"))  # "Hello"

Replacing

text = "Hello, World!"

# replace(old, new, count)
print(text.replace("World", "Python"))  # "Hello, Python!"
print(text.replace("l", "L"))           # "HeLLo, WorLd!"
print(text.replace("l", "L", 2))        # "HeLLo, World!" (first 2)

# Original unchanged (immutable!)
print(text)  # "Hello, World!"

Alignment

text = "Python"

print(text.center(20, "*"))  # *******Python*******
print(text.ljust(20, "-"))   # Python--------------
print(text.rjust(20, "-"))   # --------------Python
print(text.zfill(10))        # 0000Python (zero padding)

String Formatting

Method 1: % Operator (Old Style)

name = "Alice"
age = 25

# %s = string, %d = integer, %f = float
text = "Name: %s, Age: %d" % (name, age)
print(text)  # Name: Alice, Age: 25

price = 19.99
text = "Price: $%.2f" % price  # 2 decimal places
print(text)  # Price: $19.99

Method 2: format() Method

name = "Alice"
age = 25

# Basic
text = "Name: {}, Age: {}".format(name, age)
print(text)  # Name: Alice, Age: 25

# With index
text = "Name: {0}, Age: {1}".format(name, age)

# Named arguments
text = "Name: {n}, Age: {a}".format(n=name, a=age)

# Number formatting
price = 19.99567
text = "Price: ${:.2f}".format(price)  # Price: $20.00

Method 3: f-strings (Modern, Python 3.6+)

Most recommended!

name = "Alice"
age = 25

# Basic
text = f"Name: {name}, Age: {age}"
print(text)  # Name: Alice, Age: 25

# Expressions
print(f"Next year: {age + 1}")  # Next year: 26
print(f"Uppercase: {name.upper()}")  # Uppercase: ALICE

# Number formatting
price = 19.99567
print(f"Price: ${price:.2f}")  # Price: $19.99

num = 1000000
print(f"Number: {num:,}")  # Number: 1,000,000

ratio = 0.856
print(f"Percentage: {ratio:.1%}")  # Percentage: 85.6%

f-string Advanced Formatting

# Width and alignment
name = "Alice"
print(f"|{name:10}|")    # |Alice     | (left, width 10)
print(f"|{name:<10}|")   # |Alice     | (left explicit)
print(f"|{name:>10}|")   # |     Alice| (right)
print(f"|{name:^10}|")   # |  Alice   | (center)

# Zero padding
num = 42
print(f"{num:05}")       # 00042

# Decimal places
pi = 3.14159265359
print(f"{pi:.2f}")       # 3.14
print(f"{pi:.4f}")       # 3.1416

# Thousands separator
big = 1234567890
print(f"{big:,}")        # 1,234,567,890

String Operations

Concatenation (+)

first = "Hello"
last = "World"
full = first + " " + last  # "Hello World"

# Multiple concatenations
greeting = "Hello" + ", " + "Alice" + "!"

Repetition (*)

text = "Python "
repeated = text * 3  # "Python Python Python "

line = "-" * 50  # 50 dashes

Membership (in)

text = "Python Programming"

print("Python" in text)      # True
print("Java" in text)        # False
print("python" in text)      # False (case-sensitive)
print("gram" not in text)    # False

Comparison

# Lexicographical (alphabetical)
print("apple" < "banana")   # True
print("abc" < "abd")        # True
print("ABC" < "abc")        # True (uppercase < lowercase in ASCII)

# Case-insensitive comparison
text1 = "Python"
text2 = "python"
print(text1.lower() == text2.lower())  # True

String Iteration

Loop Through Characters

text = "Python"

for char in text:
    print(char)
# Output: P y t h o n (each on new line)

# With index
for i, char in enumerate(text):
    print(f"{i}: {char}")
# 0: P, 1: y, 2: t, ...

List Comprehension

text = "Python"

# Uppercase each character
upper_chars = [char.upper() for char in text]
print(upper_chars)  # ['P', 'Y', 'T', 'H', 'O', 'N']

# Filter vowels
vowels = [char for char in text if char.lower() in 'aeiou']
print(vowels)  # ['y', 'o']

Common String Patterns

Reversing String

text = "Python"
reversed_text = text[::-1]  # "nohtyP"

Checking Palindrome

text = "racecar"
is_palindrome = text == text[::-1]  # True

Removing Whitespace

text = "  Hello World  "
clean = text.strip()  # "Hello World"

Title Case

text = "python programming language"
title_case = text.title()  # "Python Programming Language"

Counting Words

sentence = "Python is awesome and powerful"
word_count = len(sentence.split())  # 5
