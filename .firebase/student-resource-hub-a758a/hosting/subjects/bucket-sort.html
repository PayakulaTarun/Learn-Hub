Bucket Sort

Core Computer Science • Knowledge Pillar

Theory & Concepts

Distribute elements into 'buckets' based on value range. Sort each bucket (e.g., Insertion Sort). Concatenate buckets. Great for floating point numbers 0.0 - 1.0.
