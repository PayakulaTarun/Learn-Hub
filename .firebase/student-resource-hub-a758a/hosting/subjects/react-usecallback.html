Frontend Framework • Knowledge Pillar

Theory & Concepts

Caching Functions

Every time a component re-renders, all functions inside it are recreated (new reference in memory).

The Problem

If you pass a function to a React.memo child, the child sees a 'new' prop every time and re-renders anyway, defeating memoization.

The Solution

useCallback caches the function instance itself. const handleClick = useCallback(() => { ... }, []);
