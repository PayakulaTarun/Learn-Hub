HTML Text Formatting

HTML contains several elements for defining text with a special meaning.

1. Bold and Strong

  • <b>: Defines bold text, without any extra importance.
  • <strong>: Defines text with strong importance. The content is typically displayed in bold.

Recommendation: Use <strong> when the text implies urgency or seriousness (important for screen readers). Use <b> purely for visual bolding.

2. Italic and Emphasized

  • <i>: Defines italic text (e.g., technical terms, foreign words).
  • <em>: Defines emphasized text. The content is typically displayed in italic. Screen readers stress this text.

3. Other Formatting Elements

  • <small>: Defines smaller text.
  • <mark>: Defines marked/highlighted text (yellow background usually).
  • <del>: Defines deleted text (strikethrough).
  • <ins>: Defines inserted text (underlined).
  • <sub>: Defines subscript text (e.g., H₂O).
  • <sup>: Defines superscript text (e.g., X²).
