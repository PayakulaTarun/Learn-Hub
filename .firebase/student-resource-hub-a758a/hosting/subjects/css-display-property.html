CSS Display Property

Web Development • Knowledge Pillar

Theory & Concepts

The display property is the most important CSS property for controlling layout.

  • block: Starts on a new line and takes up the full width (e.g., <div>, <h1>, <p>).
  • inline: Takes only as much width as necessary and does NOT start on a new line (e.g., <span>, <a>). Width/height properties don't work.
  • inline-block: Like inline, but you CAN set width/height.
  • none: Hides the element completely.
