Input in Python

The input() Function

Purpose: Get data from the user during program execution.

Syntax:

variable = input("prompt message")

Key Points:

  1. Always returns a string (even if user types a number)
  2. Program pauses and waits for user input
  3. User presses Enter to submit
  4. Optional prompt message

Example:

name = input("Enter your name: ")
print(name)  # Whatever user typed
print(type(name))  # <class 'str'>

Critical Beginner Mistake

❌ Common Error: Assuming input() returns the correct type

age = input("Enter age: ")  # User types: 25
future_age = age + 5  # ERROR! Can't add string and int

✅ Correct Approach: Always convert (type cast)

age = int(input("Enter age: "))  # Convert to int
future_age = age + 5  # Works!

Type Casting User Input

String to Integer

age = int(input("Enter age: "))
# User types: 25
# age is now integer 25

String to Float

price = float(input("Enter price: "))
# User types: 19.99
# price is now float 19.99

Validation Example

age_str = input("Enter age: ")
if age_str.isdigit():
    age = int(age_str)
else:
    print("Invalid age!")

Multiple Inputs

Method 1: Separate input() calls

name = input("Name: ")
age = int(input("Age: "))
city = input("City: ")

Method 2: Single line, split by space

# User types: Alice 25 NewYork
data = input("Enter name age city: ").split()
name = data[0]
age = int(data[1])
city = data[2]

Method 3: Map to convert multiple values

# User types: 10 20 30
numbers = list(map(int, input("Enter 3 numbers: ").split()))
print(numbers)  # [10, 20, 30]

Output in Python

The print() Function

Basic Syntax:

print(value1, value2, ..., sep=' ', end='
')

Parameters:

  • sep: Separator between values (default: space)
  • end: What to print at the end (default: newline)

Basic Printing

print("Hello, World!")
print(100)
print(3.14, "is pi")

Multiple Values

name = "Alice"
age = 25
print("Name:", name, "Age:", age)
# Output: Name: Alice Age: 25

Custom Separator

print("apple", "banana", "orange", sep=", ")
# Output: apple, banana, orange

print(1, 2, 3, 4, 5, sep="-")
# Output: 1-2-3-4-5

Custom End Character

print("Hello", end=" ")
print("World")
# Output: Hello World (on same line)

print("Loading", end="...")
# Output: Loading... (no newline)

String Formatting

Method 1: Concatenation (Old, Not Recommended)

name = "Alice"
age = 25
print("Name: " + name + ", Age: " + str(age))
# Problem: Must convert numbers to string

Method 2: % Operator (Old Style)

name = "Alice"
age = 25
print("Name: %s, Age: %d" % (name, age))
# %s = string, %d = integer, %f = float

Method 3: .format() Method

name = "Alice"
age = 25
print("Name: {}, Age: {}".format(name, age))
print("Name: {0}, Age: {1}".format(name, age))  # With index
print("Name: {n}, Age: {a}".format(n=name, a=age))  # Named

Method 4: f-strings (Modern, Recommended)

Introduced in Python 3.6

Syntax: Prefix string with f, use {variable} for embedding

name = "Alice"
age = 25
print(f"Name: {name}, Age: {age}")
# Output: Name: Alice, Age: 25

Why f-strings are Best:

  1. ✅ Most readable
  2. ✅ Fastest performance
  3. ✅ Can embed expressions
  4. ✅ Modern and Pythonic

f-strings Advanced Features

Expressions Inside {}

x = 10
y = 5
print(f"Sum: {x + y}")  # Sum: 15
print(f"Product: {x * y}")  # Product: 50

name = "alice"
print(f"Uppercase: {name.upper()}")  # ALICE

Number Formatting

Decimal Places:

pi = 3.14159265359
print(f"Pi: {pi:.2f}")  # Pi: 3.14 (2 decimal places)
print(f"Pi: {pi:.4f}")  # Pi: 3.1416 (4 decimal places)

Width and Alignment:

num = 42
print(f"{num:5}")    # "   42" (right-aligned, width 5)
print(f"{num:<5}")   # "42   " (left-aligned)
print(f"{num:^5}")   # " 42  " (center-aligned)
print(f"{num:05}")   # "00042" (zero-padded)

Thousands Separator:

big_num = 1000000
print(f"{big_num:,}")  # 1,000,000

Percentage:

ratio = 0.85
print(f"{ratio:.1%}")  # 85.0%

Multiline f-strings

name = "Alice"
age = 25
city = "New York"

message = f'''Name: {name}
Age: {age}
City: {city}'''
print(message)

Escape Characters

Special characters in strings:

| Escape | Meaning | Example | |--------|---------|----------| | | Newline | "Line1 Line2" | | | Tab | "Name Age" | | \\ | Backslash | "C:\\Users" | | \' | Single quote | "It\'s" | | \" | Double quote | "He said \"Hi\"" |

print("First Line
Second Line")
print("Name\tAge\tCity")
print("Path: C:\\Users\\Alice")

Raw Strings

Prefix with r to treat escape characters as literal:

path = r"C:\Users\Alice
ew_folder"
print(path)  # C:\Users\Alice
ew_folder
# Without 'r': 
 would be newline, causing error

Reading Multiple Lines

print("Enter 3 numbers:")
a = int(input())
b = int(input())
c = int(input())
print(f"Sum: {a + b + c}")

Input Validation Pattern

while True:
    try:
        age = int(input("Enter age: "))
        if age > 0:
            break
        else:
            print("Age must be positive!")
    except ValueError:
        print("Please enter a valid number!")

print(f"Your age: {age}")
