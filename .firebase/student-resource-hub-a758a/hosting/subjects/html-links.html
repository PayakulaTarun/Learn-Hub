Theory & Concepts

HTML Links

Links (Hyperlinks) are the defining feature of the Web. They allow users to click and jump from one page to another.

The <a> Tag

Links are defined with the <a> (anchor) tag. The most important attribute is href, which indicates the link's destination.

<a href="url">Link Text</a>

Types of Links

1. Absolute URLs

A full web address. Used for external websites.

<a href="https://www.google.com">Google</a>

2. Relative URLs

A local link to a file within the same website. Preferred for internal navigation (faster and works offline/dev).

<a href="about.html">About Us</a>
<a href="/images/logo.png">Logo</a>

The target Attribute

Specifies where to open the linked document.

  • _self (Default): Opens in the same window/tab.
  • _blank: Opens in a new window or tab.
<a href="page.html" target="_blank">Open in New Tab</a>

Image Links

You can use an image as a link by nesting the <img> tag inside the <a> tag.

<a href="home.html">
  <img src="logo.png" alt="Home">
</a>

Bookmarks (Internal Anchors)

You can link to a specific part of the SAME page using IDs.

Step 1: Assign an id to the target.

<h2 id="section1">Chapter 1</h2>

Step 2: Link to it using #.

<a href="#section1">Go to Chapter 1</a>
