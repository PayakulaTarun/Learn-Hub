Security in JavaScript

Web Development • Knowledge Pillar

Theory & Concepts

The web is hostile. JS code must be secure.

  • XSS: Attacker injects malicious script into your site (e.g. via comments section). Solution: Sanitize inputs, use textContent.
  • CORS: Browser blocks requests to different domains unless server allows it.
  • SOP: Scripts from one origin cannot access data from another origin.
