Theory & Concepts

HTML Drag and Drop API

HTML5 standardizes Drag and Drop functionality. Any element can be made draggable.

The Drag Process

  1. Make Draggable: Set draggable="true" on the element.
  2. Drag Start: Define what happens when dragging starts (store data).
  3. Drag Over: Allow the drop by preventing default behavior.
  4. Drop: Define what happens when dropped (read data).

Example Workflow

1. The Item

<img id="drag1" src="img.png" draggable="true" ondragstart="drag(event)">

2. The JavaScript

function drag(ev) {
  ev.dataTransfer.setData("text", ev.target.id);
}

function allowDrop(ev) {
  ev.preventDefault(); // Must do this to allow dropping
}

function drop(ev) {
  ev.preventDefault();
  var data = ev.dataTransfer.getData("text");
  ev.target.appendChild(document.getElementById(data));
}

3. The Drop Zone

<div ondrop="drop(event)" ondragover="allowDrop(event)"></div>
