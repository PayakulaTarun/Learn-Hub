Theory & Concepts

What is a Dictionary?

A dictionary is an unordered collection of key-value pairs. Each key maps to a value.

Real-Life Analogy

Think of a real dictionary (vocabulary book):

  • Word (key): "Python"
  • Definition (value): "A high-level programming language"
  • Look up word → Get definition
  • Words are unique (no duplicates)

Python dictionaries work the same:

  • Key → Value
  • Fast lookup by key
  • Keys must be unique

JSON Similarity

Python dictionaries are nearly identical to JSON objects:

JSON:

{
  "name": "Alice",
  "age": 25,
  "city": "New York"
}

Python Dictionary:

{
  "name": "Alice",
  "age": 25,
  "city": "New York"
}

Almost the same! This makes dictionaries perfect for working with JSON data.

Key Characteristics

  1. Key-Value Pairs: Data stored as key: value
  2. Keys are Unique: Duplicate keys not allowed
  3. Keys must be Immutable: str, int, tuple (not list)
  4. Values can be Anything: Any data type
  5. Unordered (Python 3.6-): No guaranteed order
  6. Ordered (Python 3.7+): Maintains insertion order
  7. Mutable: Can be modified after creation
  8. Fast Lookup: O(1) access by key

Dictionary vs List vs Tuple

| Feature | List | Tuple | Dictionary | |---------|------|-------|------------| | Syntax | [1,2,3] | (1,2,3) | {"a":1, "b":2} | | Access | Index | Index | Key | | Ordered | ✅ Yes | ✅ Yes | ✅ Yes (3.7+) | | Mutable | ✅ Yes | ❌ No | ✅ Yes | | Lookup | O(n) | O(n) | O(1) | | Use Case | Sequence | Fixed data | Key-value |

Creating Dictionaries

Method 1: Curly Braces {}

# Empty dictionary
empty = {}

# Basic dictionary
student = {
    "name": "Alice",
    "age": 20,
    "grade": "A"
}

# Different value types
person = {
    "name": "Bob",           # String
    "age": 25,               # Integer
    "height": 5.9,           # Float
    "married": False,        # Boolean
    "hobbies": ["reading"]   # List
}

Method 2: dict() Constructor

# From keyword arguments
student = dict(name="Alice", age=20, grade="A")

# From list of tuples
pairs = [("name", "Alice"), ("age", 20)]
student = dict(pairs)

# From two lists using zip
keys = ["name", "age", "grade"]
values = ["Alice", 20, "A"]
student = dict(zip(keys, values))

Method 3: Dictionary Comprehension

# Squares
squares = {x: x**2 for x in range(6)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# Conditional
evens = {x: x**2 for x in range(10) if x % 2 == 0}
# {0: 0, 2: 4, 4: 16, 6: 36, 8: 64}

Accessing Dictionary Values

Using Square Brackets []

student = {"name": "Alice", "age": 20, "grade": "A"}

# Access by key
print(student["name"])   # Alice
print(student["age"])    # 20

# KeyError if key doesn't exist
print(student["email"])  # KeyError!

Using get() Method (Safer)

student = {"name": "Alice", "age": 20}

# get() returns None if key doesn't exist
email = student.get("email")  # None

# Provide default value
email = student.get("email", "Not provided")
print(email)  # Not provided

# Existing key
name = student.get("name")
print(name)  # Alice

get() vs []:

  • dict[key]: Raises KeyError if missing
  • dict.get(key): Returns None if missing (safe)
  • dict.get(key, default): Returns default if missing

Modifying Dictionaries

Adding/Updating Values

student = {"name": "Alice", "age": 20}

# Add new key-value pair
student["grade"] = "A"
print(student)  # {'name': 'Alice', 'age': 20, 'grade': 'A'}

# Update existing value
student["age"] = 21
print(student)  # {'name': 'Alice', 'age': 21, 'grade': 'A'}

update() Method

student = {"name": "Alice", "age": 20}

# Update with another dictionary
student.update({"grade": "A", "city": "NYC"})
print(student)
# {'name': 'Alice', 'age': 20, 'grade': 'A', 'city': 'NYC'}

# Update existing and add new
student.update({"age": 21, "email": "alice@example.com"})

Removing Items

pop(key): Remove and return value

student = {"name": "Alice", "age": 20, "grade": "A"}

age = student.pop("age")  # Returns 20
print(age)  # 20
print(student)  # {'name': 'Alice', 'grade': 'A'}

# KeyError if key doesn't exist
# email = student.pop("email")  # KeyError!

# Provide default
email = student.pop("email", "Not found")
print(email)  # Not found

popitem(): Remove and return last inserted (key, value)

student = {"name": "Alice", "age": 20, "grade": "A"}

item = student.popitem()  # ('grade', 'A')
print(item)
print(student)  # {'name': 'Alice', 'age': 20}

del: Delete specific key

student = {"name": "Alice", "age": 20}

del student["age"]
print(student)  # {'name': 'Alice'}

# Delete entire dictionary
del student

clear(): Remove all items

student = {"name": "Alice", "age": 20}
student.clear()
print(student)  # {}

Dictionary Methods

Accessing Keys, Values, Items

keys(): Get all keys

student = {"name": "Alice", "age": 20, "grade": "A"}

keys = student.keys()
print(keys)  # dict_keys(['name', 'age', 'grade'])
print(list(keys))  # ['name', 'age', 'grade']

values(): Get all values

values = student.values()
print(values)  # dict_values(['Alice', 20, 'A'])
print(list(values))  # ['Alice', 20, 'A']

items(): Get all key-value pairs as tuples

items = student.items()
print(items)  # dict_items([('name', 'Alice'), ('age', 20), ('grade', 'A')])
print(list(items))  # [('name', 'Alice'), ('age', 20), ('grade', 'A')]

Other Useful Methods

copy(): Create shallow copy

original = {"name": "Alice", "age": 20}
copy = original.copy()

copy["age"] = 21
print(original)  # {'name': 'Alice', 'age': 20} (unchanged)
print(copy)      # {'name': 'Alice', 'age': 21}

setdefault(): Get value, set default if missing

student = {"name": "Alice"}

# Key exists: returns value
name = student.setdefault("name", "Unknown")
print(name)  # Alice

# Key doesn't exist: sets and returns default
age = student.setdefault("age", 18)
print(age)  # 18
print(student)  # {'name': 'Alice', 'age': 18}

fromkeys(): Create dict from sequence of keys

keys = ["name", "age", "grade"]
student = dict.fromkeys(keys, "Unknown")
print(student)  # {'name': 'Unknown', 'age': 'Unknown', 'grade': 'Unknown'}

# Without default (None)
student = dict.fromkeys(keys)
print(student)  # {'name': None, 'age': None, 'grade': None}

Iterating Over Dictionaries

Iterate Over Keys (Default)

student = {"name": "Alice", "age": 20, "grade": "A"}

# Default iteration is over keys
for key in student:
    print(key)
# Output: name, age, grade

# Explicit
for key in student.keys():
    print(key)

Iterate Over Values

for value in student.values():
    print(value)
# Output: Alice, 20, A

Iterate Over Key-Value Pairs (Best)

for key, value in student.items():
    print(f"{key}: {value}")
# Output:
# name: Alice
# age: 20
# grade: A

Accessing Values During Iteration

scores = {"math": 90, "science": 85, "english": 92}

# Method 1: Using items()
for subject, score in scores.items():
    print(f"{subject}: {score}")

# Method 2: Using keys
for subject in scores:
    score = scores[subject]
    print(f"{subject}: {score}")

Nested Dictionaries

Dictionaries within dictionaries (like nested JSON).

Creating Nested Dictionaries

# Student database
students = {
    "student1": {
        "name": "Alice",
        "age": 20,
        "grades": {"math": 90, "science": 85}
    },
    "student2": {
        "name": "Bob",
        "age": 22,
        "grades": {"math": 75, "science": 88}
    }
}

Accessing Nested Values

# Access outer key
student1 = students["student1"]
print(student1)  # {'name': 'Alice', 'age': 20, 'grades': {...}}

# Access nested key
name = students["student1"]["name"]
print(name)  # Alice

# Access deeply nested
math_score = students["student1"]["grades"]["math"]
print(math_score)  # 90

Iterating Nested Dictionaries

for student_id, info in students.items():
    print(f"
{student_id}:")
    print(f"  Name: {info['name']}")
    print(f"  Age: {info['age']}")
    print("  Grades:")
    for subject, score in info['grades'].items():
        print(f"    {subject}: {score}")

Output:

student1:
  Name: Alice
  Age: 20
  Grades:
    math: 90
    science: 85

student2:
  Name: Bob
  Age: 22
  Grades:
    math: 75
    science: 88

Dictionary Lookup Efficiency

Why Dictionaries are Fast

Dictionaries use hash tables for O(1) average lookup!

List (Linear Search - O(n)):

my_list = [("name", "Alice"), ("age", 20), ("grade", "A")]

# Find value for "age"
for key, value in my_list:
    if key == "age":  # Checks each item
        print(value)
# Time: O(n) - slow for large data

Dictionary (Hash Lookup - O(1)):

my_dict = {"name": "Alice", "age": 20, "grade": "A"}

# Find value for "age"
print(my_dict["age"])  # Instant!
# Time: O(1) - very fast!

Performance Comparison

| Operation | List | Dictionary | |-----------|------|------------| | Access by key/index | O(1) | O(1) | | Search value | O(n) | O(n) | | Insert | O(n) | O(1) | | Delete | O(n) | O(1) | | Membership test | O(n) | O(1) |

When to use dictionary:

  • Need fast key-based lookup
  • Data has natural key (ID, name, etc.)
  • Frequent insertions/deletions
  • Representing structured data (like JSON)

Dictionary Comprehension

Syntax: {key_expr: value_expr for item in iterable}

# Squares
squares = {x: x**2 for x in range(6)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# From lists
names = ["Alice", "Bob", "Charlie"]
ages = [25, 30, 35]
people = {name: age for name, age in zip(names, ages)}
# {'Alice': 25, 'Bob': 30, 'Charlie': 35}

# With condition
even_squares = {x: x**2 for x in range(10) if x % 2 == 0}
# {0: 0, 2: 4, 4: 16, 6: 36, 8: 64}

# Transform existing dict
scores = {"math": 85, "science": 90, "english": 78}
passed = {subject: score for subject, score in scores.items() if score >= 80}
# {'math': 85, 'science': 90}

Checking Keys/Values

Check if Key Exists

student = {"name": "Alice", "age": 20}

# Using 'in'
if "name" in student:
    print("Name exists")

if "email" not in student:
    print("Email doesn't exist")

Check if Value Exists

if "Alice" in student.values():
    print("Alice is a value")

if 20 in student.values():
    print("20 is a value")
