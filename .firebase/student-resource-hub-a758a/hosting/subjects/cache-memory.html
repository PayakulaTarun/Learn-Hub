Cache Memory

Core Computer Science • Knowledge Pillar

Theory & Concepts

Small, fast memory between CPU and RAM. Stores frequently used data. Hit = Found in Cache (Fast). Miss = Go to RAM (Slow). Avg Access Time = HitTime + MissRate * MissPenalty.
