Theory & Concepts

What are Attributes?

Attributes provide additional information about HTML elements.

  • All HTML elements can have attributes.
  • Attributes are always specified in the start tag.
  • Attributes usually come in name/value pairs like: name="value".

Common Attributes

1. href Attribute

Used in <a> tags to specify the URL of the page the link goes to.

<a href="https://www.google.com">Visit Google</a>

2. src Attribute

Used in <img> tags to specify the path to the image to be displayed.

<img src="image.jpg">

3. width and height Attributes

Specify the size of images (in pixels).

<img src="img.jpg" width="500" height="600">

4. alt Attribute

Specifies an alternative text for an image if the image cannot be displayed (crucial for accessibility).

<img src="img.jpg" alt="A cute cat">

5. style Attribute

Used to add styles to an element, such as color, font, size, etc.

<p style="color:red;">This is red text.</p>

6. lang Attribute

Used in the <html> tag to declare the language of the webpage.

<html lang="en">

7. title Attribute

Defines some extra information about an element. usually shown as a tooltip when the mouse moves over the element.

<p title="I am a tooltip">Hover over me!</p>

Quoting Attribute Values

Typically, we use double quotes. Single quotes can also be used, but double quotes are the standard convention.
