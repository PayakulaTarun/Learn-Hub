Theory & Concepts

Making it Fast

Performance tuning involves optimizing hardware, configuration, schema, and queries.

The EXPLAIN Command

Prefix any SELECT statement with EXPLAIN to see how MySQL executes it. Look for:

  • type: ALL is bad (Full Table Scan). ref or range is good (Index usage).
  • rows: How many rows scanned.
  • key: Which index was used.

Slow Query Log

A log file that records queries taking longer than long_query_time (e.g., 2 seconds). Analyzing this log helps identify bottlenecks.
