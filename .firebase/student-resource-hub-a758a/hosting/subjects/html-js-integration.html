Theory & Concepts

Integration with JavaScript

JavaScript provides interactivity. It connects to HTML via the <script> tag.

1. External Script (Best Practice)

<script src="app.js"></script>

It is best to place this:

  • At the end of the body (</body>), OR
  • In the head with the defer attribute.

2. Internal Script

<script>
  alert('Hello');
</script>

3. Inline Events (Avoid)

HTML elements have event attributes like onclick, onmouseover.

<button onclick="doSomething()">Click</button>

Better approach: Select the element in JS and add an event listener.

The <noscript> Tag

Defines content to be displayed if the user has disabled JavaScript.

<noscript>Please enable JavaScript to use this site.</noscript>

Manipulation (DOM)

JavaScript sees HTML as a tree of objects (DOM). It can change content (innerHTML), styles (style), and attributes.
