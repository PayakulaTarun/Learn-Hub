Bootstrap Accessibility (ARIA)

Frontend Framework • Knowledge Pillar

Theory & Concepts

Inclusive Design

Bootstrap aims for WCAG 2.1 AA compliance.

Interactive Elements

  • Modals/Dropdowns: Automatically handle aria-expanded, aria-hidden via JS.
  • Focus: Bootstrap manages focus traps (keeping tab focus inside invalid modals).

Helpers

  • .visually-hidden: Hides text visually but keeps it available for Screen Readers (e.g., 'Skip to content' links).
